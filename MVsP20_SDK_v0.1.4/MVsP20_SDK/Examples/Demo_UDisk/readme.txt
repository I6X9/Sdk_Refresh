Demo_UDisk工程主要演示了FATFS文件系统在P20芯片上的使用。


硬件环境要求:
    1. P20系列开发板
    2. 外接串口小板，TX/RX/GND 接至 C2/C3/GND 引脚 (波特率 115200)

    SD卡接线：
    	 SD_DAT0   b[20]
    	 SD_CLK    b[21]
    	 SD_CMD    b[22]
    USB接线：
         DM        a[30]
         DP        a[31]
         
         
软件使用说明:
    系统初始化完成之后，会直接进入到while循环
    1.检测U盘是否插入
    2.在U盘插入的情况下，查找是否存在Udisk.txt文件，没有文件则创建文件，可对其进行读写操作
    3.查找U盘中是否存在.MVA格式文件（代码升级包），如存在该格文件，则将文件转存FLASH中进行双BANK升级操作，此时芯片将会运行新的程序
 
  
至于如何制作MVA升级包会另有工具生成，升级注意事项会另有相关文档说明，这里不再赘述，工程中制作一个简单的U盘双BANK升级演示，具体注意点视客户需求而定。
