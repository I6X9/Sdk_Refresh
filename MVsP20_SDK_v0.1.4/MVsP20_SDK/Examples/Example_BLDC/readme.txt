BLDC工程主要演示了P20系列芯片atimer驱动电机的相关功能，以下是对应代码环境简介：
硬件环境要求:
    1、P20系列开发板（有专用电机接线口）
    2、电机驱动小板
    3、外接串口小板或者直接使用P20系列开发板上的串口，TX/RX/GND 接至 C2/C3/GND 引脚

接线说明：驱动板跳帽选择HALL信号：HA、HB、HC

         P20开发板         电机驱动小板      电机
                         + + + + + + + + 
          B6    <--->    + DHA         +          
          B5    <--->    + DLA         +
PWM       B4    <--->    + DHB         +
          B3    <--->    + DLB         +
          B2    <--->    + DHC         +
          B1    <--->    + DLC         +
BREAK_IN  B0    <--->    + V-T         +
                         +             +
                         +          5V + <---> 5V
          A16   <--->    + FB-A     HA + <---> HU
HALL      A17   <--->    + FB-B     HB + <---> HV
          A18   <--->    + FB-C     HC + <---> HW
          GND   <--->    + GND      GND+ <---> GND
                         +             +
                         +           A + <---> U
电机接口                 +           B + <---> V
                         +           C + <---> W
                         + + + + + + + +

软件使用说明:
    1、系统上电之后会通过串口打印提示选择，设置电机的启停、速度(pwm占空比)、故障复位等；
    2、串口输入‘R’则电机启动
    3、串口输入‘S’则电机停止
    4、串口输入‘F’则有故障时复位系统故障,无故障时为无效指令
    5、串口输入‘0~9’则改变PWM占空比，0~9分别对应5%~~95%
    6、电机运行时，串口会定时打印输出当前PWM占空比及HALL状态值

注意：
    1、为防止电机烧坏，试运行时可以将PWM设置小一点(30%以下)
    2、注意HALL信号不要接错
    3、电机UVW顺序不要接错（有的电机标记的是ABC）      
    4、如果报OC过流故障，可减小PWM、调节电机驱动小板滑动电阻值、检测硬件电路是否损坏
