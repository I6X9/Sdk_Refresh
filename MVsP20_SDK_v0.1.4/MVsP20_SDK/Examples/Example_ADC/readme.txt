adc_example工程主要演示了P20系列芯片的ADC模块的相关功能。


硬件环境要求:
    1. P20系列开发板
    2. 外接串口小板，TX/RX/GND 接至 C2/C3/GND 引脚 (波特率 115200)
    3. GPIOA3接线到J11上

软件使用说明:
	1. 上电启动之后，自动采集GPIOA3的相对电压，并通过UART输出；
    2. 可操作KEY2-KEY11，改变GPIOA3的相对电压值。

说明：
    1、DC采样和AC采样只能二选一，不可同时进行。
    2、DC模式下根据需要选择参考电压源和时钟频率。
    3、AC模式下只能采样mic数据，需要注意采样率和主时钟问题，和DC模式下是有区别的。
