Demo_RTC工程演示了P20 RTC日历和闹钟功能。

硬件环境要求:
	1.P20系列开发板。
	2.外接串口小板，TX/RX/GND 接至 C2/C3/GND 引脚(波特率 115200)。

软件环境要求:
	1.RTC包含农历和闹钟，两者都可以配置。
	2.设置好RTC变量sRtcControl和闹钟变量sNvmRtcInfo后，首先需要调用的函数就是RTC_Init。
	3.闹钟支持单次闹钟、每天一次闹钟、每周一次闹钟、工作日闹钟、用户自定义闹钟等五种，可以根据产品需要来选择。
		单次闹钟：只在特定时刻响应一次。
		每天一次闹钟：每天特定时刻都可以响应一次。
		每周一次闹钟：每周特定时刻都可以响应一次。
		工作日闹钟：周一到周五的特定时刻都可以分别响应一次。
		用户自定义闹钟：用户设置的周一到周日的特定时刻都可以分别响应一次。
	4.一般日历和闹钟设置的最小单位是分钟，Demo_RTC最小单位是秒仅仅是为了方便调试。实际使用设置时，闹钟的秒请直接设置为0，日历的秒请直接设置为当前秒数。
	5.闹钟标志的获取可以采取中断方式或轮循方式。
