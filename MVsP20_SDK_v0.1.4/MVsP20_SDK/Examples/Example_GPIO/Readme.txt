GPIO_Example工程主要演示了P2 GPIO的基本使用方法。

硬件环境要求:
    1. P2系列开发板
    2. 外接串口小板，TX/RX/GND 接至 C2/C3/GND 引脚 (波特率 115200)


软件使用说明:
	上电启动之后，可通过串口输入不同指令，进入3种例程；
	
	1. 输入‘o' 进入 Output Example：
	
	接A2至示波器，分别输入‘h’或者‘l’，观察示波器电平变化

	2. 输入‘i’，进入input Example：
	
	将A3接至A2，分别输入‘h’或者‘l’，观察打印A3 High或者 A3 Low

	3. 输入‘t’，进入Interrupt Example：
	
	将A4接接至A2，

	3.1 先配置中断类型，输入‘p'或者‘n’或者‘e’，分别代表 上升沿、下降沿、边沿（即上升沿与下降沿）触发
	3.2 分别输入‘h’或者‘l’，观察打印结果。
