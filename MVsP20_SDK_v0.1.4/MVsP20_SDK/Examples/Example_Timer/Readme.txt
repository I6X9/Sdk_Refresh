Timer_Example工程主要演示了P2 Timer的基本使用方法。

硬件环境要求:
    1. P2系列开发板
    2. 外接串口小板，TX/RX/GND 接至 C2/C3/GND 引脚 (波特率 115200)


软件使用说明:
	上电启动之后，可通过串口输入不同指令，进入6种例程；
	1. 输入‘a’，进入Timer2的中断模式例程；
	2. 输入‘A’，进入Timer2的查询模式例程；
	3. 输入‘b’，进入Timer3的中断模式例程；
	4. 输入‘B’，进入Timer3的查询模式例程；
	5. 输入‘c’，进入PWM简单应用例程；（复杂应用例程参考PWM Example）
	6. 输入‘d’，进入PWC简单应用例程；（复杂应用例程参考PWC Example）
